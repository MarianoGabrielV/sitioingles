# Sitio Web Clases de Ingles

Este repositorio incluye dos versiones del mismo sitio promocional:

- `react-site/` (React + Vite)
- `vue-site/` (Vue + Vite)

## Requisitos

- Node.js 18+
- npm 9+

## Ejecutar version React

```bash
cd react-site
npm install
npm run dev
```

## Ejecutar version Vue

```bash
cd vue-site
npm install
npm run dev
```

## Build

En cualquiera de las dos carpetas:

```bash
npm run build
```
